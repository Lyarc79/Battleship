# BattleShip
A classic Battleship game built with **Vanilla JavaScript (ES6+)**, **Webpack**, and **Jest**. The project focuses on Object-Oriented Programming (OOP) and Test-Driven Development (TDD) to separate the core game logic—such as hit tracking, ship states, and board validation—from the DOM UI. It features player vs. AI turn-based gameplay, automated fleet positioning, and full unit test coverage using `jest`.
